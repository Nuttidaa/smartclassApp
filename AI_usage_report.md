#AI usage report
##AI tools 
used in this project included GitHub Copilot (GPT-5.3-Codex) for code generation, refactoring.
##AI generated 
the main Flutter project structure, screen files, navigation flow, pastel-blue minimal UI theme, form logic, and integrations for GPS (geolocator), QR scanning (mobile_scanner), and local storage (shared_preferences).
##Manual implementation 
included creating and configuring the Firebase project in Firebase Console, running CLI commands (flutter pub get, flutter build web, firebase init, firebase deploy), and validating deployment results in the browser. Additional manual work involved handling environment-specific setup (Windows Developer Mode, permissions) and final testing/verification of app behavior.
Overall, AI accelerated development and documentation, while project configuration, deployment actions, and runtime validation were completed manually.
